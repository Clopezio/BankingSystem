# Text Editor Project

This project is a simple text editor application built with TypeScript. It allows users to open, edit, and save text files.

## Project Structure

```
text-editor-project
├── src
│   ├── index.ts          # Entry point of the application
│   ├── components
│   │   └── editor.ts     # Contains the Editor class for text file operations
│   └── utils
│       └── fileOperations.ts # Utility functions for file operations
├── package.json          # NPM configuration file
├── tsconfig.json         # TypeScript configuration file
└── README.md             # Project documentation
```

## Features

- Open text files
- Edit content
- Save changes to text files

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd text-editor-project
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage

To start the application, run:
```
npm start
```
